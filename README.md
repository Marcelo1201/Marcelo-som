# Marcelo-som
# Sobre o projeto
Link para a demo https://marcelo-som.vercel.app/

Um projeto criado para aprendizagem do "pré-processador" do css, o SASS.

Uma loja virtual sobre guitarras, onde possui aulas e guitarras à venda.
